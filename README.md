# NetSniff

**NetSniff** is a hobby Android app built with Kotlin and Jetpack Compose. Its goal is to help discover nearby devices on the same network or via Bluetooth.

> ⚠️ This app is purely for educational and personal experimentation purposes. It does not attempt to perform any intrusive actions and respects user permissions and privacy.

---

## ✨ Features

- 🔍 Scan for devices connected to the same local network
- 📡 Detect nearby Bluetooth devices (with permission)
- 💡 Simple, modern UI with Compose
- ⏳ Animated scan button and visual feedback

---

## 🛠 Tech Stack

- **Kotlin**
- **Jetpack Compose**
- **Material 3**
- **Coroutines**
- **Android Permissions API**

## 🚀 Getting Started

1. Clone the repository
2. Open in Android Studio
3. Set minimum SDK to `26+`
4. Run on a real device (Bluetooth/Wi-Fi features may not work on emulator)

---

## 📱 Permissions

This app may request the following runtime permissions:

- `ACCESS_FINE_LOCATION` – Required for Bluetooth discovery
- `BLUETOOTH` / `BLUETOOTH_SCAN` – For Bluetooth scanning
- `ACCESS_WIFI_STATE` / `INTERNET` – For network scanning
